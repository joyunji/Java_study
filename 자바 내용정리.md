# 0. JVM이란?
## JVM(Java Virtual Machine:자바 가상 머신)
    java byte code를 os에 맞게 해석 해주는 소프트웨어
### java byte code
    JVM에서만 실행되는 어떤 cpu와도 관계없는 바이너리 코드(binary code)
